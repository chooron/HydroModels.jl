## ✨ 核心功能

### 1. D8 流向分析 ⭐⭐⭐

**函数:** `compute_d8_flow_direction(dem::Raster)`

**功能:**
- 根据 DEM 自动计算 D8 流向
- 使用与 HydroRoute 兼容的编码（1, 2, 4, 8, 16, 32, 64, 128）
- 自动处理 NoData 值和边界条件
- 考虑对角线距离的正确坡度计算

**实现细节:**
- 检查 8 个邻居的高程差
- 计算坡度（考虑正交和对角线距离）
- 选择最大坡度方向作为流向
- 边界和无效值用 0 表示

---

### 2. 位置提取 ⭐⭐

**函数:** `extract_positions_from_raster(raster, valid_mask)`

**功能:**
- 从栅格中提取有效像元的位置索引
- 支持自定义验证函数
- 返回 (row, col) 位置列表

**用途:**
- 为 `build_aggr_func` 提供位置参数
- 筛选特定条件的像元（如高程范围）

---

### 3. 栅格重采样 ⭐⭐

**函数:** `match_raster_to_dem(raster, dem, method)`

**功能:**
- 将任意栅格重采样到 DEM 的分辨率和范围
- 支持多种插值方法：
  - `:nearest` - 最近邻（适合分类数据）
  - `:bilinear` - 双线性插值（适合连续数据）
  - `:cubic` - 三次插值（更平滑）

**用途:**
- 匹配降雨、土地利用等数据到 DEM
- 数据融合和预处理

---

### 4. HydroRoute 快速构建 ⭐⭐⭐

**函数:** `create_hydroroute_from_dem(dem, name)`

**功能:**
- 一键从 DEM 提取所有 HydroRoute 组件
- 返回流向矩阵、位置、聚合函数、HRU 类型

**输出:**
```julia
(
  flow_dir = flow_direction_matrix,
  positions = valid_positions,
  aggr_func = aggregation_function,
  hru_types = hru_type_vector
)
```

## 🔧 技术实现

### D8 流向编码

```
   32  64  128     西北  北  东北
    ↖  ↑  ↗         ↖   ↑   ↗
16 ← • → 1      西 ←  中心  → 东
    ↙  ↓  ↘         ↙   ↓   ↘
   8   4   2      西南  南  东南
```

### 与 HydroRoute 的集成

流向矩阵可以直接用于 `HydroModels.build_aggr_func`：

```julia
flow_dir = compute_d8_flow_direction(dem)
positions = extract_positions_from_raster(dem)
aggr_func = HydroModels.build_aggr_func(flow_dir, positions)

# 用于 HydroRoute
@hydroroute :my_route begin
    fluxes = begin
        @hydroflux outflow ~ k * storage
    end
    dfluxes = begin
        @stateflux storage ~ inflow - outflow
    end
    hru_types = ones(Int, length(positions))
    aggr_func = aggr_func  # 使用生成的聚合函数
end
```

## 🚀 使用流程

### 完整工作流程

```julia
# 1. 加载包
using HydroModels
using Rasters

# 2. 读取 DEM
dem = Raster("watershed_dem.tif")

# 3. 计算流向
flow_direction = compute_d8_flow_direction(dem)

# 4. 提取位置
positions = extract_positions_from_raster(dem)

# 5. 创建聚合函数
aggr_func = HydroModels.build_aggr_func(flow_direction, positions)

# 6. 匹配其他数据
rainfall = Raster("rainfall.tif")
rainfall_matched = match_raster_to_dem(rainfall, dem)

# 7. 构建 HydroRoute
# （需要定义 fluxes 和 dfluxes）
```
